# 🌿 Aarogyavardhini Wellness Website

A wellness-focused website built using **HTML, CSS, and JavaScript**, deployed live using **Netlify** with a custom **.life domain**.

---

## Project Overview

This project demonstrates the complete workflow of building a static website and deploying it live with a custom domain.
The website is designed with a clean and user-friendly interface focused on wellness.

---

## Tech Stack

* HTML
* CSS
* JavaScript
* Git & GitHub
* Netlify (Deployment)
* Hostinger (Domain management)

## 🌐 Step-by-Step Process

### 1️⃣ Website Development

* Built a static website using HTML, CSS, and JavaScript
* Structured files into organized folders
* Designed UI and added interactivity

---

### 2️⃣ Version Control

* Initialized Git repository
* Committed project files
* Pushed code to GitHub

---

### 3️⃣ Domain Setup

* Purchased a custom domain (**aarogyavardhini.life**)
* Configured domain settings

---

### 4️⃣ Deployment

* Connected GitHub repository to Netlify
* Deployed the website
* Generated a live project URL

---

### 5️⃣ Domain Integration

* Linked custom domain with deployed site
* Verified domain connection

---

### 6️⃣ Final Setup

* Enabled HTTPS (SSL)
* Website made publicly accessible
